# Barbell Diva v142

Cartella pronta per GitHub Pages.

## Pubblicazione

1. Crea o apri un repository GitHub.
2. Carica tutto il contenuto di questa cartella nella root del repository.
3. In **Settings → Pages**, scegli `Deploy from a branch`.
4. Seleziona il branch `main` e la cartella `/root`.
5. Salva e apri l’indirizzo GitHub Pages dopo qualche minuto.

La pagina principale è `index.html`. Il file `.nojekyll` evita problemi con gli asset della PWA.
